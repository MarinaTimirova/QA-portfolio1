**Область тестирования**

**В тестирование входят**
- Главная страница сайта
- Регистрация пользователя
- Авторизация
- Восстановлене пароля
- Каталог товаров
- Поиск товаров
- Карточка товаров
- Добавление товаров в корзину
- Изменение количества товаров
- Удаление товаров из корзины
- Личный кабинет
- Оформление заказа
- Навигация по сайту

  **в тестирование НЕ входят**
  - Нагрузочное тестирование
  - Производительность
  - Безопасность
  - Совместимость с большим количеством браузеров
  - Автоматизированое тестирование
 
    **Типы тестирования**
    - Функциональное
    - Smoke
    - Regression
    - UI testing
    - Usability testing
    - Positive testing
    - Negative testing

      **Тетовое окружение**
      - ОС: Windows 10 Pro
      - Браузер: 26.4.5.646 (64-bit)
      - Интернет: Стабильное подключение
     
**Критерии начала тестирования**
- Сайт доступен.
- Все страницы открываются.
- Сервер отвечает без ошибок
  
**Критерии завершения тестирования**
- Выполнены все тест-кейсы.
- Все найденные дефекты зарегистрированы.
- Подготовлен итоговый отчёт.
**Риски**
- Возможны временные проблемы с доступностью демо-сайта.
- Некоторые функции могут быть ограничены, так как это демонстрационный проект.
